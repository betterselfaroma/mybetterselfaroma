# Production Test Checklist

Use this checklist before and after deploying the Admin PWA to `https://scentknowsyou.com`.

## 1. Supabase SQL

Run these SQL files in Supabase SQL Editor, in order, if they have not already been applied:

1. `supabase/migrations/0008_admin_pwa_mobile_support.sql`
2. `supabase/migrations/0009_admin_pwa_security_hardening.sql`

Both files are safe for existing data: they use `CREATE ... IF NOT EXISTS` / `ALTER TABLE ... ADD COLUMN IF NOT EXISTS`, do not drop tables, and include:

```sql
NOTIFY pgrst, 'reload schema';
```

## 2. Promote Admin Account

Replace the email before running:

```sql
UPDATE customers
SET role = 'admin', is_admin = true
WHERE email = 'YOUR_EMAIL@example.com';
```

If an older database does not have `customers.is_admin`, use:

```sql
UPDATE customers
SET role = 'admin'
WHERE email = 'YOUR_EMAIL@example.com';
```

## 3. Vercel Environment Variables

Set these in the production hosting environment:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `ADMIN_EMAILS`
- `NEXT_PUBLIC_SITE_URL=https://scentknowsyou.com`

Security checks:

- Do not prefix `SUPABASE_SERVICE_ROLE_KEY` with `NEXT_PUBLIC_`.
- Do not expose the service role key in client components.
- Do not upload `.env.local`; local env files are ignored by `.gitignore`.

## 4. PWA Files

After deploy, confirm these URLs return `200`:

- `https://scentknowsyou.com/manifest.webmanifest`
- `https://scentknowsyou.com/service-worker.js`
- `https://scentknowsyou.com/icons/icon-192.png`
- `https://scentknowsyou.com/icons/icon-512.png`
- `https://scentknowsyou.com/.well-known/assetlinks.json`

Manifest requirements:

- `start_url` is `/admin`
- `scope` is `/admin`
- `display` is `standalone`
- `theme_color` is `#2f5d46`
- `background_color` is `#f7f1e6`

## 5. HTTPS Admin Smoke Test

On desktop browser:

- [ ] `/admin` can open.
- [ ] Logged-out users are redirected away from Admin.
- [ ] Normal members cannot enter `/admin`.
- [ ] Admin / staff users can enter `/admin`.
- [ ] `/admin/bookings` can read bookings.
- [ ] `/admin/members` can read members.
- [ ] `/admin/points` can read point transactions.
- [ ] Booking Confirm works.
- [ ] Booking Complete works.
- [ ] Booking Cancel works.
- [ ] WhatsApp buttons open `https://wa.me/60...` links correctly.

## 6. Mobile Scan Test

On a real phone over HTTPS:

- [ ] `/admin/scan` opens for admin / staff only.
- [ ] Normal members cannot enter `/admin/scan`.
- [ ] Camera permission is requested only after tapping the scan button.
- [ ] Camera opens successfully.
- [ ] Scanning a member QR displays the member profile.
- [ ] Scanning stops after success.
- [ ] The rescan button works.
- [ ] `/admin/scan?token=REAL_TOKEN` displays the member profile.
- [ ] A full QR URL such as `https://scentknowsyou.com/member/checkin?token=REAL_TOKEN` works.
- [ ] A plain token value works.
- [ ] Invalid QR shows `二维码无效`.
- [ ] Unknown token shows `找不到会员`.

## 7. Points Test

- [ ] Add 10 points succeeds.
- [ ] Add 50 points succeeds.
- [ ] Redeem / deduct 10 points succeeds.
- [ ] Custom point adjustment succeeds.
- [ ] Points cannot go below zero by default.
- [ ] `points_transactions` has a new row for each adjustment.
- [ ] `admin_audit_logs` has a new row for each admin action.

## 8. Add to Home Screen

On mobile browser:

- [ ] `/admin` shows the install hint:
  `想像手机 App 一样使用？点击浏览器菜单 → Add to Home Screen / 添加到主屏幕。`
- [ ] Browser menu allows Add to Home Screen.
- [ ] Installed icon appears on the phone.
- [ ] Opening the installed app starts directly at `/admin`.
- [ ] The app runs in standalone mode.

## 9. Debug APK Readiness

Before Bubblewrap debug APK:

- [ ] HTTPS Admin PWA tests above pass.
- [ ] `https://scentknowsyou.com/manifest.webmanifest` is live.
- [ ] `https://scentknowsyou.com/service-worker.js` is live.
- [ ] `docs/android-apk-build.md` has Bubblewrap setup and build steps.
- [ ] `public/.well-known/assetlinks.json` exists.

For a verified TWA, replace the TODO SHA256 fingerprint in `assetlinks.json` with the debug or release signing certificate fingerprint generated by Android tooling.
