# Claude Code Master Prompt｜Better Self Aroma / 闻见更好的自己

你现在接手的是一个 Next.js + Tailwind CSS 的双语 Landing Page 项目。

## 项目目标
建立一个高转化网页，推广两个服务套餐：

1. RM60｜香气状态查看体验
2. RM150｜专属特调精油提升方案

核心不是卖普通精油，而是卖：
精神价值 × 精油香气 × 自我提升

中文名：
闻见更好的自己

英文名：
Better Self Aroma

## 技术要求
- Next.js App Router
- Tailwind CSS
- TypeScript
- 手机端优先
- 中文 / 英文双语切换
- 默认中文
- 不需要后端
- 不需要付款系统
- CTA 全部导向 WhatsApp
- WhatsApp 电话先使用 placeholder：60123456789

## 你需要做的事
1. 运行项目并检查页面是否正常。
2. 保持当前文案逻辑和安全边界。
3. 优化 UI 细节、间距、动效、移动端体验。
4. 保持价格展示策略：
   - 首屏只轻轻显示 “RM60 起 / from RM60”
   - 正式价格在套餐区域展示
5. 不要把网站做成价目表。先让用户感受到价值，再展示价格。
6. 不要加入医疗承诺，不要说治疗焦虑、失眠、疾病。
7. 不要做玄学风，不要塔罗、水晶球、星座盘。
8. 图片真实素材没准备前，使用 /public/images/ 里的 SVG 占位图。
9. 以后真实图片替换时，保持同名文件即可。

## 成交逻辑
- 首屏：卖感觉和价值
- 痛点：让用户觉得“这就是我”
- 理念：解释香气不是治疗，而是进入状态的入口
- 套餐：RM60 建立信任，RM150 带走完整方案
- 流程：让用户知道钱花在哪里
- FAQ：处理心理咨询 / 医疗功效 / 升级 RM90 的疑虑
- CTA：引导 WhatsApp 预约

## 禁止内容
不要出现：
- 治疗焦虑
- 治疗失眠
- 治愈创伤
- 改变命运
- 预测运势
- 医疗诊断
- 心理咨询替代
- 玄学符号
- 水晶球、塔罗、星盘
